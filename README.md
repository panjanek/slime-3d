# Slime simulation in 3d

Implementation of "slime" in 3D.
Each agent (glowing dot) moves forward, initially at random. Agents sense proximity do other agents with one sensor in front and turn towards (or away) other agents.
Sensitivity, forces and randomization can be configured.
